# GazeViz 1.0

Gaze Tracking Visualization Tool

### Prerequisites

```
Python 2
Tkinter
PIL Image, ImageDraw, ImageTk, ImageOps
os 
```

## Instructions

```
python GazeViz.py path/to/image.png path/to/data.txt
```

Example
```
python GazeViz.py images/example_image.JPG data/example_data.txt
```
